# Checker Informations

Checker is a simple and open-source program written in Python for PSN account checking. **This program is only working on a Mac OS**. Feel free to change its original purpose and port it to other platforms.


# Usage

Run `validator.py` with python and select your mail:pass combo list. The program will output mail:pass:connection-type list. After that use your new combo with`PSN.py` to output list with accounts with games.
